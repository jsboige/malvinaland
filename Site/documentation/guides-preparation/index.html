🏗️ Guides de Préparation des Mondes

Guides détaillés pour préparer chaque monde du jeu de piste Malvinaland

ℹ️ À propos de ces guides

Ces guides vous aideront à préparer chaque monde de Malvinaland pour une session de jeu. Chaque guide contient :

  • Description détaillée du monde
  • Matériel nécessaire
  • Instructions de préparation
  • Conseils pour l'animation
  • Points de sécurité à vérifier

🚀 Prochaines étapes

Une fois que vous avez consulté les guides de préparation :

  1. Consultez le Manuel Administrateur pour la vue d'ensemble
  2. Référez-vous au Guide PNJ pour l'animation
  3. Utilisez le Guide de Résolution en cas de problème
